Le code utilisé emploie trois programmes différents :

 - zero.py comprend la calibration Nelson Seigel et tout ce qui se rapporte à la courbe
   zéro coupon (forward, prix ZC, etc.)
 - cir.py comprend la calibration au CAP et des interfaces pour simuler les trajectoires
   CIR++.
 - mbs.py comprend les fonctions pour calculer les diverses valeurs ayant rapport à
   l'évaluation du MBS et des durées.


Pour évaluer ces programmes, vous aurez besoin de Python 3 ainsi que de quelques packages
scientifiques standard (essentiellement numpy, scipy, pandas et matplotlib pour la
production de graphiques).
